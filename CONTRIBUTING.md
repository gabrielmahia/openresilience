# Contributing to OpenResilience

Accuracy, safety, and harm minimization take precedence over feature velocity.

## Before Contributing
1. Open an Issue describing the problem and impact
2. Await discussion before submitting PR

## Accepted Contributions
- Data adapters
- Scoring improvements
- UI accessibility
- Security hardening
